# Custom Rooms — Discord-бот

Бот для персональных приватных голосовых «комнат»: каждый пользователь может создать
свою роль с цветом и приватный голосовой канал, в который попадают только приглашённые им
участники. Остальных бот отключает из голосового.

## Возможности

- `/room create <название> <цвет>` — создаёт роль с указанным цветом **прямо над** якорной
  ролью, выдаёт её автору и заводит комнату в базе. Одна комната на пользователя.
  Цвет: hex (`#ff0000` / `ff0000`) либо имя (`Red`, `Blue`, `Green`, …).
- `/room manage` — меню выбора пользователей; выбранным выдаётся роль комнаты и они
  заносятся в whitelist (базу).
- **Триггер-канал** (join-to-create): при входе в него
  - владелец → создаётся приватный голосовой `・<название>`, владелец перемещается внутрь;
  - участник из whitelist → перемещается в активную комнату владельца;
  - кто не в базе → бот сразу отключает (disconnect) из голосового.
- Пустой временный голосовой канал автоматически удаляется.

## Настройка

1. **Установка зависимостей:**
   ```bash
   npm install
   ```

2. **Токен** уже лежит в `.env` (`DISCORD_TOKEN`). ⚠️ Токен был отправлен открытым текстом —
   настоятельно рекомендуется **перевыпустить** его в Developer Portal → Bot → Reset Token и
   обновить `.env`. Файл `.env` уже в `.gitignore`.

3. **Привилегированные intents** (Developer Portal → Bot → Privileged Gateway Intents):
   включить **Server Members Intent** (и Voice State работает по умолчанию).

4. **Права бота на сервере:** Manage Roles, Manage Channels, Move Members, View Channels,
   Connect. Роль бота должна быть **выше** якорной роли `1507662794387095562`.

5. **ID** в `config.js`:
   - `ANCHOR_ROLE_ID` — якорная роль (`1507662794387095562`);
   - `TRIGGER_CHANNEL_ID` — триггер-канал (`1513276381025271808`).
   - Опционально `GUILD_ID` в `.env` — для мгновенной регистрации команд на одном сервере.

## Запуск

```bash
node deploy-commands.js   # один раз: регистрация слэш-команд
node index.js             # запуск бота
```

## Структура

| Файл | Назначение |
|------|-----------|
| `config.js` | Константы: токен, ID ролей/каналов, префикс |
| `db.js` | SQLite (better-sqlite3): комнаты и whitelist |
| `commands/room.js` | Слэш-команда `/room` (create, manage) |
| `deploy-commands.js` | Регистрация команд в Discord |
| `index.js` | Клиент и события (ready, interaction, voiceStateUpdate) |
